# lr-landing

This is a project to create basic landing pages for Learning Registry resources

## Dependencies

Mongrel2

Brubeck

Redis

MongoDB

## Starting

### Mongrel2

From the mongrel2 directory run the following commands

    m2sh start -db the.db -host localhost
    m2sh start -db the.db -host localhost
    
### Brubeck

From the src directory run

    ./landing.py
